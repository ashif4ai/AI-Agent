## First Time Setup
If this is your first time on this project:
1. Read `RULES.md` completely
2. Read `PROJECT_CONTEXT.md`
3. Read `MASTER_PROGRESS.md`
4. Check `MEMORY.md` for past decisions
5. Go to current phase folder and read `PHASE_STATUS.md`
6. Open current task file

## Returning to Project
If resuming work:
1. Read `RULES.md` (refresh the rules)
2. Check `MASTER_PROGRESS.md` for current status
3. Review last 5 entries in `MEMORY.md`
4. Go to current task in current phase
5. Check if task is "In Progress" or "Testing"
